Digit皮肤
========

Digit 是一组适合在平板和手机上面使用的组件。 你可以选择以下三种皮肤：flat，light和dark。

## Digit更新
在组件模板中默认会使用Digit组件。组件的皮肤正在开发中。为了确保你使用的是最新版本，请按照以下步骤配置你的项目：

2. 在命令行中切换到项目的当前目录。
	
		cd yourprojectfolder
		
3. 运行以下命令

	npm install digit@latest --save
	
--save 参数指定把digit包加入到项目依赖文件package.json中。
